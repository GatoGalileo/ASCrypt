AS2 library:

* The project has been developed with FlashDevevelop IDE and MTASC.

AS3 library:

* The project has been developed with FlashDevevelop IDE and Flex SDK.

PHP library:

* Tested only with PHP5, not sure does it work with PHP4.
* MD5, RIPEMD-160, SHA1 and SHA-256 requires hash and hash_hmac functions.
* AES-128/192/256 and ARC4 requires the mcrypt library.

CSharp library:

* The library requires Microsoft.NET 2.0 framework.
* The project has been developed with Visual C# 2008 Express.
